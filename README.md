# Mate

## Run locally
- `cd mate`
- Make sure to have the `.env` file inside `mate` directory.
- `python3 -m venv .venv`
- `source .venv/bin/activate`
- `pip install -r requirements.txt`
- `python3 manage.py makemigrations`
- `python3 manage.py migrate`
- `python3 manage.py runserver`
- `http://127.0.0.1:8000/`
